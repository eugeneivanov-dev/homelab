# Infrastructure Overview

This repository documents the infrastructure of my personal homelab environment.

The lab is used to practice and experiment with systems administration, networking, virtualization, and infrastructure architecture.

## Architecture

The homelab is organized into several logical layers:

Network Layer  
Routing, switching, and structured cabling.

Compute Layer  
Workstations and future lab nodes used for virtualization and experiments.

Storage Layer  
Network storage used for backups and lab services.

Power Layer  
Rack UPS and power distribution for equipment protection.
