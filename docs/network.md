# Network Infrastructure

This homelab network is built to simulate a small infrastructure environment for learning networking and systems administration.

## Core Components

#### Router  
UDM Pro Max  
Provides routing, firewall, DHCP and network management.

#### Switch  
UniFi Switch  
Handles internal network switching for lab devices.

#### Structured Cabling  
CAT6 patch panel with short patch cables used for clean cable management.

## Network Design

The network follows a simple layered structure:

Internet  
↓  
Router (UDM Pro Max)  
↓  
Switch (UniFi)  
↓  
Internal Devices

## Goals

- Practice network configuration
- Experiment with infrastructure services
- Maintain stable home network environment
